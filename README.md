# Kanchi
感知: RSS + RabbitMQ

## Getting Started

```
cp config/config-template.toml config/config.toml
cp config/history-template.json config/history.json
npm i
npm start
```

## Manually publishing to RabbitMQ

```
cp config/config-template.toml config/config.toml
cp config/manual-template.toml config/manual.toml
npm i
npm run manual
```
